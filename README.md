## Objetivo del proyecto

El objetivo principal del sistema es administrar libros digitales de manera organizada y eficiente, además de mantener un inventario consistente para garantizar la trazabilidad de usuarios y transacciones.

Como objetivos específicos, el sistema permite realizar búsquedas rápidas, gestionar usuarios y roles, controlar préstamos digitales y generar informes simples.

## Funcionalidades principales

Este Sistema de Gestión de Libros Electrónicos permite administrar de forma eficiente una biblioteca digital mediante el control de usuarios y contenidos.

Entre sus principales funcionalidades se encuentran:

- Registro e inicio de sesión de usuarios.
- Gestión de usuarios y asignación de roles (Administrador y Lector).
- Administración de libros electrónicos (agregar, editar y eliminar).
- Organización de libros por categorías, autores y fechas de publicación.
- Almacenamiento de archivos digitales en formatos como PDF.
- Búsqueda avanzada de libros.
- Visualización y lectura de libros en línea.
- Descarga de libros según los permisos del usuario.
- Control de acceso para proteger el contenido digital.
- Generación de informes y estadísticas sobre libros registrados, usuarios activos y títulos más consultados.
